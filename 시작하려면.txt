참고) 나중에 다시 전체 서버를 실행하시려면?
VSC(Visual Studio Code)나 터미널을 열고 E:\OneDrive\바탕 화면\LIMS 폴더 안에서 다음 두 가지를 각각 실행해주시면 됩니다.

1. 프론트엔드 (UI 화면) 실행

npm run dev

2. 백엔드 (하드게이트 및 모의 API) 실행 Express 서버 구동을 위해 필요한 모듈을 최초 1회 설치하고 실행합니다.

npm install express cors
node server/index.js

(이후 백엔드는 http://localhost:4000에서 백그라운드로 실행되며 API 요청을 처리합니다!)
지금 바로 웹 브라우저(http://localhost:5173)를 열어 완성된 KAIC 디자인과 상호작용 가능한 무결성 폼들을 클릭해 보세요!
